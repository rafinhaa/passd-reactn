## 🏁 How to run the project
#### Clone the repository
```bash
git clone https://github.com/rafinhaa/passd-reactn
cd passd-reactn
```

#### Install dependencies
```bash
yarn install
cd ios && pod install && cd ..
```

#### Start server
```bash
yarn run ios
```
